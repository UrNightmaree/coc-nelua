# coc-nelua

Nelua language server extension for coc.nvim

## Install

`:CocInstall coc-nelua`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-nelua-keymap)`

## Lists

`:CocList demo_list`
